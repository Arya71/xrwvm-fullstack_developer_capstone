About Us

Welcome to the Car Dealership Review Application. This platform allows users to explore different car dealerships, read customer reviews, and share their own experiences about car purchases. Our mission is to provide transparent and helpful information to customers so they can make informed decisions when buying vehicles.

The application is built using modern web technologies including Django, React, and IBM Cloud services. It integrates dealership data, customer reviews, and sentiment analysis to improve the overall user experience.
